Data Upload and Excel Merging using Flask Application

About Data Upload and Excel Merging

  • Data upload is a tool designed to upload an excel sheet
  • After uploading, one can view the content in the webpage
  • To uplaod the first excel file to the database, click on "Upload"
  • In order to merge any excel file to the first file, please choose file and upload, then click on "merge"
  • Merged file can be viewed by clicking "merge"
